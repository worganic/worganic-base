Composant <wo-tchat-ia>

Composant Angular 18 standalone de tchat IA, entièrement réutilisable hors contexte Frankenstein. Toutes les URLs et options sont configurables via @Input(). Supporte le streaming SSE, les formulaires dynamiques, la sélection de zones, l'historique multi-sessions et bien plus.

open_in_full 3. Usage complet 

<wo-tchat-ia
            [visible]="showChat"
            executorUrl="http://localhost:3002"
            apiUrl="http://localhost:3001"
            provider="claude"
            model="claude-sonnet-4-6"
            title="Assistant Projet"
            systemPrompt="Tu es expert en gestion de projet agile..."
            initialPrompt="Bonjour ! Quel est votre besoin ?"
            [context]="{ user: currentUser, siteName: 'MonApp', params: { role: 'admin' } }"
            [allowFormGeneration]="true"
            [allowZoneSelection]="true"
            [allowMarkdown]="true"
            [allowExport]="true"
            [allowContinueAfterEnd]="false"
            [quickPrompts]="['Reformule', 'Plus de détails', 'Résume']"
            endMarker="[FIN_TCHAT]"
            [formTemplates]="myForms"
            (closed)="onChatClosed($event)"
            (aiResponse)="onAiMsg($event)"
            (formSubmitted)="onForm($event)"
            (messageSent)="onUserMsg($event)"
            (zoneModificationRequested)="onZone($event)">
          </wo-tchat-ia>

dns 5. API executor requise — POST /execute-chat-turn

Le composant envoie ce payload et attend un flux SSE en réponse.

                "provider": "claude",
                "model": "claude-sonnet-4-6"
              }
